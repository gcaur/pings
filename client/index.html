Welcome to the Internet Connectome project

To efficiently connect people with each other, it would be very useful to have the ability to estimate ahead of time the internet delay between different internet addresses.
Using the "PING" applet, you can help us construct a connectome of the internet, i.e. a topology of network connectivity.
As a bonus after you have collected 500+ connections, we will tell you where you fit according to our model.
The project originates from a collaboration between Ubisoft and the machine learning lab (LISA) at Université de Montréal.
The data will be made publically and freely available to all. We target potential applications in advanced match-making, video on demand and multiplayer game experience and we use machine learning to predict delays between yet unmeasured pairs of addresses.

Start pinging now! *

When you start the applet:

1. Make sure to enter a valid identifier (nickname) in order to appear in the leaderboard.
2. Let the applet run for a while. The more data (about ping delays between your machine and other randomly chosen addresses) you help us collect, the better!
3. After 500 connections from all around the world, we will display a detailed analysis of your Internet access.



* This Java applet is digitally signed by the Université de Montréal. Click "Run" to authorize access to network functions. No personal data will be collected.

Need help?

As our applet is still in a prototypical stage, bugs may occur.
Don't hesitate to contact us at iconnectome@googlegroups.com if you have any question or suggestion.

If you encounter a reproducible bug, you can help us fix it by issuing the following command:

firefox http://iconnect.iro.umontreal.ca/applet_jar.html &> ping.log

and send us the resulting ping.log file along with information about your OS, browser and Java environment.
