Start pinging now! *


The applet starts collecting data automatically.
1. Make sure to enter a valid identifier (nickname) in order to appear in the leaderboard.
2. Let the applet run for a while by keeping this browser window open. The more data (about ping delays between your machine and other randomly chosen addresses) you help us collect, the better!
3. After you have reached 500 connections, we will display a detailed analysis of your Internet connection according to our preliminary model.


* This Java applet is digitally signed by the Université de Montréal. Click "Run" to authorize access to network functions. No personal data will be collected.

Welcome to the Internet Connectome project

To optimize peer-to-peer internet connections, it would be very useful to have the ability to estimate ahead of time the internet delay between different internet addresses.
Using the "PING" applet, you can help us construct a connectome of the internet, i.e. a topology of network connectivity.
As a bonus after our applet has collected 500+ connections, we will provide a detailed analysis of your Internet connection according to our preliminary model.
The project originates from a collaboration between Ubisoft and the machine learning lab (LISA) at Université de Montréal.
The data will be made publicly and freely available to all. We target potential applications in matchmaking, video on demand and multiplayer game experience and we use machine learning to predict delays between yet unmeasured pairs of addresses.

Need help?

As our applet is still in a prototypical stage, bugs may occur.
Don't hesitate to contact us at iconnectome@googlegroups.com if you have any question or suggestion.

If you encounter a reproducible bug, you can help us fix it by issuing the following command (under MacOS and Linux):

firefox http://iconnect.iro.umontreal.ca &> ping.log

and send us the resulting ping.log file along with information about your OS, browser and Java environment.
