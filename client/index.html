Start pinging now! *


The applet starts collecting data automatically.
1. Make sure to enter a valid identifier (nickname) in order to appear in the leaderboard.
2. Let the applet run for a while by keeping this browser window open. The more data (about ping delays between your machine and other randomly chosen addresses) you help us collect, the better!
3. After the applet has run for 15 minutes, we will display a detailed analysis of your Internet connection according to our preliminary model.


* This Java applet is digitally signed by the Université de Montréal. Please authorize the applet as it needs access to network functions. No personal data will be collected.

Troubleshooting

As our applet is still in a the prototype stage, bugs may occur.
Don't hesitate to contact us at iconnectome@googlegroups.com if you have any questions or suggestions.


If the applet does not start:
Your browser should automatically initiate the installation of the required Java version (> 1.6) or redirect you to the appropriate installation page for your OS/browser. Restarting your browser may be required after installation. You can verify your installation on the Java tester page, which should display Java Version 1.6 or above.


If Java cannot be installed successfully:
Java can be tricky to install on certain system architectures, OS or browser. Please follow installation instructions for your specific setup by searching Google. You can verify your installation on the Java tester page, which should display Java Version 1.6 or above.


If Java starts but you encounter a reproducible bug in the applet:
Please run your browser session within a console and send us the resulting output along with information about your OS, browser and Java environment.

  • Under Windows, you need to enable the "Java console". In the Control Panel, search for Java and start the Java configuration software. Click on the advanced tab. In the "Java console", check show the console option. Restart your browser and reload this web page. A window will open with the output of the applet.
  • Under Mac OS X, open system preferences. Find the Java preferences. Open the advanced preferences tab and select the option to show the console. Restart your browser and reload this web page. A window will open with the output of the applet.
  • Under Linux, simply launch your browser from the terminal with the following command:

<YOUR_BROWSER> http://iconnect.iro.umontreal.ca

where <YOUR_BROWSER> is your browser executable, e.g. firefox, google-chrome, etc.

The Internet Connectome project

To optimize peer-to-peer internet connections, it would be very useful to have the ability to estimate ahead of time the internet delay between different internet addresses.
Using the "PING" applet, you can help us construct a connectome of the internet, i.e. a topology of network connectivity.
As a bonus after our applet has run for 15 minutes, we will provide a detailed analysis of your Internet connection according to our preliminary model.
The project originates from a collaboration between Ubisoft and the machine learning lab (LISA) at Université de Montréal.
The data will be made publicly and freely available to all. We target potential applications in matchmaking, video on demand and multiplayer game experience and we use machine learning to predict delays between yet unmeasured pairs of addresses.
