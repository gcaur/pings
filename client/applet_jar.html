The Internet Connectome Project Test Page

Instructions

  • Fill the name box with a nick name. The nickname is used to collect the effort statistics. Use a common nickname if you want to form teams (be careful, it's case-sensitive!).
  • Let it run for a while. Let the applet run for a while so that it can gather lots of information. The more we have, the better!
  • Check the leaderboard!
  • A signed applet. The applet is digitally signed by the Université de Montréal so that Java can access the platform-specific ping and traceroute commands.
  • Help us debug! This applet is still in its prototypical stages, and bugs may occur. One way to trap Java's debug output is to redirect both standard output and standard error to a file, using, for example, a command-line such as

               firefox http://www.stevenpigeon.org/secret/applet_jar.html &> ping.debug.log
    You can send us (at pigeon@iro.umontreal.ca) the log if something goes wrong, or if you have suggestions, questions, etc.
